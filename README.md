Expense Control

Sistema completo para controle de despesas e receitas pessoais.

Projeto fullstack desenvolvido com **.NET + React + TypeScript**

---

Tecnologias

Backend
- .NET 8
- ASP.NET Web API
- Entity Framework Core
- SQLite
- JWT Authentication

Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- AG Grid
- Axios

---

Funcionalidades

- Cadastro e login de usuários
- Autenticação com JWT
- CRUD de pessoas
- CRUD de categorias
- Registro de receitas e despesas
- Totais por pessoa/categoria

---

Autenticação

- Login com JWT
- Token armazenado no localStorage
- Endpoints protegidos
